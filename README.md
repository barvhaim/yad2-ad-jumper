# yad2-ad-jumper
Yad2 ad jumper
## Getting Startedt
Configure your credentials and link to your ad in creds.js


### Prerequisites
npm install puppeteer
npm install node-cron
npm install pm2 -g

## Running the tests
run once: 

```
node index.js
```

Pm2 will now start the app.js

```
pm2 start index.js
```
